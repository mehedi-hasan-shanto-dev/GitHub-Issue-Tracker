1️⃣ What is the difference between var, let, and const?

Ans.
var can't maintain block scope. let/const can maintain block scope.
function/local scope -> var/let/const can't use outside of the function.
Using var we can declare multiple variable in one name.
can't redeclare with same name by let and const.
var and let we can reassign value. but const can't reassign value.

2️⃣ What is the spread operator (...)?

Ans.
The spread operator ... is used to copy, combine, or expand arrays and objects easily, without writing long code.

3️⃣ What is the difference between map(), filter(), and forEach()?

Ans.
map(map = transform) → makes a new array with changed values.

filter(filter = select) → makes a new array but only with items that pass a test.

forEach(forEach = just do) → just loops through items and does something, no new array.

Ans.
Normal function = full sentence.
Arrow function = shortcut version of the same sentence written with =>.Arrow function is quick,simple and easy to read.

5️⃣ What are template literals?

Ans.
Template literals use backticks (`).
They put variables inside strings with ${ }.They make multi-line strings simple.No need for + to join words, smart strings.
